This project develops ShareCraft, a skill-sharing platform in the domain of online education and peer-to-peer learning. Users act as both students and tutors. They register and authenticate, create and manage bookings for tutoring sessions, exchange messages linked to bookings, search for tutors via profiles, skills, and reviews, and use admin capabilities for reports, moderation, and blocked content.

Role

Fullstack developer

Tools

IntelliJ, VSCode, Docker, JMeter, GitHub

Focus

Architecture, maintainability, usability, and design pattern implementation
